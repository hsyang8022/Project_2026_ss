# EO/SAR 영상 기반 지상 표적 자동탐지 및 분석 지원 서비스 기능명세서 및 유저 시나리오

## 1\. 서비스 개요

### 1.1 서비스명

EO/SAR 영상 기반 지상 표적 자동탐지 및 분석 지원 서비스

### 1.2 서비스 목적

- EO·SAR 위성영상에서 지상 표적 후보를 탐지·분류
- AI 탐지 결과의 신뢰도와 판단 근거를 함께 제공
- 영상판독관이 탐지 결과를 검토하고, 재확인 대상과 판독 우선순위를 판단할 수 있도록 지원
- 분석 결과, 판독관 의견, 시간대별 변화 내역을 로그로 저장하여 교대 인수인계와 후속 분석에 활용

### 1.3 서비스 설계 원칙

- AI는 최종 판단자가 아니라 판독 보조 도구로 사용
- 탐지 결과는 “표적 확정”이 아니라 “표적 후보”로 표시
- 화력유도 자동화가 아니라, 판독관의 재확인 우선순위 판단을 지원
- 실시간 추적이 아니라, 시간대별 위성영상 비교를 통한 변화 분석으로 제한
- Grad-CAM과 SHAP은 AI 판단 근거를 설명하기 위한 보조 지표로 활용

### SAR 분석 시 주의사항

- SAR 영상은 색상보다 밝기값, 반사강도, 그림자, 산란점 중심으로 해석
- 탐지 결과는 차량 후보이며, 실제 표적 확정으로 표시하지 않음
- SAR 노이즈 또는 촬영 조건으로 인한 오탐 가능성을 신뢰성 검토와 연결

| 조건  | 판독 권고 |

## 시나리오 1. 영상판독관의 EO 영상 빠른 분석

### 사용자

영상판독관

### 상황

판독관은 새롭게 수집된 EO 위성영상을 확인해야 한다.  
기존에는 이미지를 직접 확대하며 객체 후보를 찾았지만, 본 서비스에서는 AI 탐지 결과를 먼저 확인하고 재검토 대상을 선별한다.

### 수행 흐름

1.  사용자가 로그인한다.
2.  메인 대시보드에서 오늘의 분석 대기 영상을 확인한다.
3.  EO 분석 페이지로 이동한다.
4.  사이드바에서 EO 원본 영상을 선택한다.
5.  AI 모델이 영상 내 객체 후보를 탐지한다.
6.  화면에 bbox, 클래스명, 신뢰도가 표시된다.
7.  사용자는 신뢰도가 낮거나 유사 클래스 혼동 가능성이 있는 객체를 선택한다.
8.  신뢰성 검토 페이지에서 Grad-CAM overlay를 확인한다.
9.  AI 집중 영역이 객체 내부에 있는지 확인한다.
10. 판독관 의견을 입력하고 분석 결과를 저장한다.

### 결과

- 탐지 객체 후보가 정리됨
- 재확인 대상이 선별됨
- 판독관 의견이 로그로 저장됨
- 교대자 또는 후속 분석자가 결과를 확인할 수 있음

## 시나리오 2. SAR 영상에서 차량 후보 탐지 및 세부 분류

### 사용자

영상판독관, 데이터 사이언티스트

### 상황

SAR 광역 영상에서 차량 후보가 있는지 확인해야 한다.  
SAR 영상은 EO 영상보다 직관적 해석이 어렵기 때문에 AI 탐지 결과와 신뢰성 검토가 필요하다.

### 수행 흐름

1.  사용자가 SAR 분석 페이지로 이동한다.
2.  SAR 원본 영상을 선택한다.
3.  모델이 광역 SAR 영상에서 차량 후보 위치를 탐지한다.
4.  탐지된 객체를 crop하여 분류 모델에 입력한다.
5.  차량 후보 클래스와 신뢰도가 출력된다.
6.  사용자는 탐지 결과를 확인한다.
7.  신뢰성 검토 페이지에서 SAR 영상의 AI 집중 영역을 확인한다.
8.  SHAP 기여도에서 노이즈, 텍스처, 객체 크기 등이 판단에 미친 영향을 확인한다.
9.  신뢰도가 낮거나 설명 가능성 결과가 불안정한 객체를 재확인 대상으로 표시한다.
10. 판독관 의견을 저장한다.

### 결과

- SAR 영상의 차량 후보 탐지 결과 확보
- 신뢰성 낮은 객체를 재확인 대상으로 분류
- SAR 판독 부담 완화
- 분석 결과가 로그로 저장됨

## 시나리오 3. 시간대별 변화 분석을 통한 신규·소실 객체 확인

### 사용자

영상판독관, 교대자

### 상황

같은 지역에 대해 10:00 영상과 10:30 영상이 존재한다.  
실시간 추적은 불가능하지만, 두 시점의 위성영상을 비교하여 새로 등장한 객체와 사라진 객체를 확인해야 한다.

### 수행 흐름

1.  사용자가 비교 분석 페이지로 이동한다.
2.  비교 모드에서 “시간대별 변화 분석”을 선택한다.
3.  이전 시점 영상과 현재 시점 영상을 선택한다.
4.  각 시점 영상에 대해 AI 탐지를 수행한다.
5.  시스템은 이전 객체와 현재 객체를 bbox 중심좌표, 클래스, 거리 기준으로 비교한다.
6.  객체별 상태를 유지, 신규, 소실, 위치 변화, 클래스 변화, 불확실로 분류한다.
7.  화면에는 이전 영상, 현재 영상, 이전 탐지 결과, 현재 탐지 결과가 4분할로 표시된다.
8.  변화가 큰 객체는 재확인 필요로 표시된다.
9.  판독관은 변화 분석 결과를 검토하고 의견을 작성한다.
10. 변화 분석 결과를 로그로 저장한다.

### 결과

- 이전 시점 대비 신규 등장 객체 확인
- 소실 객체 확인
- 위치 변화 후보 확인
- 재확인 필요 객체 선별
- 교대 인수인계 자료로 활용 가능

## 시나리오 4. 단일·지역·광역표적 후보 분류

### 사용자

영상판독관, 관리자

### 상황

AI 탐지 결과가 단순 객체 목록으로만 제공되면 판독 우선순위를 정하기 어렵다.  
따라서 탐지 결과를 단일표적 후보, 지역표적 후보, 광역표적 후보로 구분한다.

### 수행 흐름

1.  사용자가 분석 결과가 있는 영상을 선택한다.
2.  시스템은 탐지 객체 수, bbox 위치, 객체 간 거리, 시간대별 반복 탐지 여부를 확인한다.
3.  객체가 하나일 경우 단일표적 후보로 표시한다.
4.  일정 구역에 여러 객체가 밀집된 경우 지역표적 후보로 표시한다.
5.  여러 구역 또는 여러 시점에서 반복 변화가 나타나는 경우 광역표적 후보로 표시한다.
6.  사용자는 표적 후보 유형별 판독 권고를 확인한다.
7.  단일표적 후보는 정밀 확인 대상으로 검토한다.
8.  지역표적 후보는 구역 단위 재확인 대상으로 검토한다.
9.  광역표적 후보는 지속 감시 및 우선순위 검토 대상으로 분류한다.
10. 결과를 저장한다.

### 결과

- 단순 탐지 결과가 판독 목적에 맞게 정리됨
- 판독관이 어떤 대상을 먼저 볼지 판단하기 쉬워짐
- 시간대별 변화 분석과 연결 가능
- 대시보드의 군사적 맥락이 강화됨

## 시나리오 5. 신뢰성 검토를 통한 재확인 대상 선별

### 사용자

영상판독관, 데이터 사이언티스트

### 상황

AI가 특정 객체를 높은 신뢰도로 탐지했지만, 실제로 객체를 보고 판단했는지 배경을 보고 판단했는지 확인해야 한다.

### 수행 흐름

1.  사용자가 EO 또는 SAR 분석 결과에서 특정 객체를 선택한다.
2.  신뢰성 검토 페이지로 이동한다.
3.  선택 객체의 confidence, class, bbox 정보를 확인한다.
4.  Grad-CAM 결과를 확인한다.
5.  AI 집중 영역이 객체 내부에 있는지 확인한다.
6.  Grad-CAM이 blank로 표시되면 설명 불가 상태로 판단한다.
7.  SHAP 기여도에서 노이즈, 텍스처, 객체 크기, 영상 품질 요인을 확인한다.
8.  시스템은 신뢰, 재확인, 수동 판독 필요 중 하나의 권고 상태를 표시한다.
9.  사용자는 최종 판독 의견을 입력한다.
10. 결과를 로그에 저장한다.

### 결과

- AI 탐지 결과의 설명 가능성 확인
- 배경 의존 탐지 또는 노이즈 영향 탐지 식별 가능
- 재확인 대상 선별 가능
- 모델 개선 필요 조건 도출 가능

## 시나리오 6. 교대 근무자의 인수인계 확인

### 사용자

교대자, 영상판독관

### 상황

근무 교대 시 이전 근무자가 분석한 영상과 재확인 필요 객체를 빠르게 확인해야 한다.

### 수행 흐름

1.  교대자가 로그인한다.
2.  메인 대시보드에서 오늘의 분석 요약을 확인한다.
3.  분석 로그 / 인수인계 페이지로 이동한다.
4.  시간, 지역, 센서종류, 재확인 필요 여부로 필터링한다.
5.  이전 근무자가 남긴 판독 의견을 확인한다.
6.  신규 등장, 소실, 위치 변화 객체를 우선 확인한다.
7.  필요 시 원본 영상과 분석 결과를 다시 연다.
8.  추가 판독 의견을 작성한다.
9.  인수인계 상태를 완료로 표시한다.

## 시나리오 7. 데이터 담당자의 모델 성능 및 취약 조건 확인

### 상황

### 수행 흐름

1.  데이터 담당자가 로그인한다.
2.  신뢰성 검토 페이지에서 모델 성능 요약을 확인한다.

4.  클래스별 성능 또는 탐지 실패 사례를 확인한다.
5.  SHAP 기여도 히스토그램을 확인한다.
6.  노이즈, 객체 크기, 텍스처 균질성, 대비 저하 등이 성능에 미친 영향을 확인한다.
7.  재학습 필요 조건을 정리한다.
8.  필요한 경우 분석 로그를 CSV로 Export한다.

- 모델 취약 조건 파악
- 재학습 데이터 확보 방향 설정

# 8\. 최종 판단

본 서비스는 단순 객체탐지 모델 시연이 아니라, 영상판독관이 AI 탐지 결과를 검토하고 재확인 우선순위를 판단할 수 있도록 돕는 판독 보조 서비스로 설계한다.

핵심 기능은 다음과 같다.

- EO/SAR 영상 기반 객체 탐지
- 탐지 결과의 신뢰성 검토
- Grad-CAM 기반 AI 집중 영역 확인
- SHAP 기반 판단 조건 기여도 확인
- 시간대별 변화 분석
- 단일·지역·광역표적 후보 분류
- 판독관 의견 저장
- 분석 로그 및 인수인계 관리

단, 본 서비스는 실시간 추적, 화력유도 자동 판단, 특정 화력장비 추천, 현재 위치 확정 기능은 제공하지 않는다. 모든 결과는 판독관의 최종 판단을 보조하기 위한 참고 정보로 제한한다.
